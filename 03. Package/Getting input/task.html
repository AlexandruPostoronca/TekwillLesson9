Getting input

The Scanner class is widely used to parse text to strings, and it is by far the simplest way to get input in Java.

In this task, you simply need to implement the getString() method of InputReader to read a whole line of input using the nextLine() method of Scannerand return it as String. The trick is to use the Scanner class without the import statement.

The application will receive a string from the getString() method and output it in the corresponding format.

Sample Input 1: 

Test

Sample Output 1: 

The input string: Test
